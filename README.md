# Project Kupi-i-Vozi

"Kupi i vozi" je veb aplikacija koja omogućava pretragu postojećih oglasa polovnih automobila prema raznim kriterijumima, registraciju korisnika i postavljanje sopstvenog oglasa, čuvanje oglasa u favoritima, obaveštavanje korisnika putem e-pošte o novom oglasu i sl.
